# 中国电影票房爬虫 (2024)

这是一个基于 Python 的简单爬虫项目，用于抓取 [BoxOfficeCN](http://www.boxofficecn.com/boxoffice2024) 网站上 2024 年中国大陆电影的票房数据。通过该脚本可以获取每部电影的排名、上映年份、电影名称和票房（万元），并将结果保存为 CSV 文件，方便后续数据分析或存档。

## 功能特点

- 自动抓取 2024 年所有上映电影的票房信息（包括重映影片）。
- 处理了页面中部分年份被 `<span>` 标签包裹的特殊情况（如重映影片用红色标记年份）。
- 对电影名称进行简单清洗（移除中文左括号）。
- 输出为结构化的 CSV 文件，可直接用 Excel 或数据分析工具打开。

## 技术栈

- **Python 3.x**
- **requests** – 发送 HTTP 请求，获取网页 HTML。
- **lxml** – 解析 HTML，使用 XPath 提取数据。
- **re** – 正则表达式，用于字符串清洗（可选，实际可用字符串替换代替）。
